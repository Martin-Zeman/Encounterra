# Encounterra
Combat encounter simulator for D&D 5e

The software architecture aims to achieve the following objectives:

- Ensure modularity of abilities (making the assignment of abilities to any character straightforward).
- Provide effortless scalability for incorporating new abilities and mechanics.
- Ensure the autonomy of combat sessions, allowing them to operate independently of each other.
- Enable efficient support for parallel execution of processes.
- Facilitate seamless integration with machine learning techniques, such as reinforcement learning.
